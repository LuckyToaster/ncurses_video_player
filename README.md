# TERM_PLAY
Reproduce any video on the terminal

## Dependencies
* Linux system
* gcc
* ffmpeg
* ncurses

## Install 
- ```git clone https://github.com/LuckyToaster/ncurses_video_player.git```
- ```make``` To compile

## Run
* ```make run <args...>``` to run with make
* Or by running the executable ```./termplay```
